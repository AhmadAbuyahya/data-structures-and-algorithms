# Stacks and Queues
Implementation of the Stack and Queue data structures in Javascript 
## Challenge
Implementation of the Stack and Queue data structures in Javascript 

## Approach & Efficiency
approach: array implementation
Efficiency: O(1) for all methods so far

## API
Stacks:
.push(value): adds a value to the top of the stack.
.pop(): removes the value at the top of the stack.
.peek(): returns the value at the top of the stack.
.isEmpty(): checks wether or not the stack is empty.

Queues:
.Enqueue(value): adds a value to rear of the queue.
.Dequeue(): removes the value ata the front of the queue.
.peek(): returns the value at the front of the queue.
.isEmpty(): checks wether or not the queue is empty.
